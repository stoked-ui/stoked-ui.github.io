  • Stoked UI vs. Base UI

    Stoked UI implements Google's Material Design whereas Base UI features many of the same components, but without the Material Design implementation.

  • Does it support Material Design 3?

    The adoption of Material Design 3 is tentatively planned for Stoked UI v7. See the the release schedule and follow this GitHub issue for future updates.
